# Module Boundary Examples

This file provides worked examples that illustrate where information belongs
within the WFF module structure. It is intended to help practitioners and
agents correctly assign content to the right module.

## Example 1: Footpad dermatitis in broiler chickens

| Module | What belongs here |
|--------|------------------|
| I — Zootechnical Description | Litter type, stocking density, ventilation, litter management practices |
| II — Veterinary Inventory | Footpad dermatitis as a biological consequence; severity grading scale |
| III — Affective Quantification | Pain intensity category per lesion grade; estimated duration; valence (negative) |
| IV — Epidemiological Review | Prevalence of each lesion grade in the target production system |
| V — Econometric Calculation | Cumulative Pain from footpad dermatitis, by intensity category, per production cycle |
| VI — Expression and Notation | Welfare footprint expression for footpad dermatitis contribution |

**Key boundary to note:** The lesion grade (Module II) is a biological consequence.
The pain intensity category (Module III) is the affective experience. These are
different quantities requiring different evidence and different module placement.
